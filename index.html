WiSDOM: WriterScript based Data Obfuscation Module

Created during HackDSC'21 (Google DSC's national level Hack-a-thon) under Open Innovation in Data Privacy and Security domain. Secured second position.


WriterScript

Word Count dependent Esoteric Programming Language based on BrainF*ck Logic.


xSFOS

(X Six Four Operating System) is a simple 64bit Linux Kernel from scratch. I am making this project as a teaching tool. The aim is to create well-structured documentation and proper releases on Git so that anyone can trace my steps and learn the same concepts.


xSimpleLinuxKernelModule

Project to understand more about Linux Kernel Modules development.


NS3 Cybersecurity Simulations

Collection of cybersecurity simulations in NS3 and C++ for academia. Some of the simulations are build from scratch and some are build upon existing solutions. With common aim to aid in understanding Cybersecurity Concepts and be able to play with them.


SAMPARK : Website Security Scanning Framework

National Hack-a-thon project (Won 1st Rank in Information Security Domain)


Android Device Threat Report Generation using PACE framework and ADB

Semester 2 Research Project, CSDF-VITB


Android Permission Extraction and Dataset Creation with Python

Project module for PACE and PACER (our research papers).


Movement Tracing in Android using Polygraph lines.

Project in association with Madhya Pradesh Police Department (India) under CSDF Department, VIT Bhopal.
