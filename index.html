xSFOS

(X Six Four Operating System) is a simple 64bit Linux Kernel from scratch. I am making this project as a teaching tool. The aim is to create well-structured documentation and proper releases on Git so that anyone can trace my steps and learn the same concepts.


xSimpleLinuxKernelModule

Project to understand more about Linux Kernel Modules development.


NS3 Cybersecurity Simulations

Collection of cybersecurity simulations in NS3 and C++ for academia. Some of the simulations are build from scratch and some are build upon existing solutions. With common aim to aid in understanding Cybersecurity Concepts and be able to play with them.
